# Formulário de Login 🔒

Este é um projeto simples de um **formulário de login**, desenvolvido com HTML e CSS.
## 🚀 Tecnologias

* **HTML:** Estrutura semântica do formulário.
* **CSS:** Estilização, layout e responsividade.
* **Font Awesome:** Ícones de usuário e cadeado.
